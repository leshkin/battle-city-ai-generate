# Comparison of AI models for creating an analogue of the game Battle City (1985)

## Initial conditions

Only one prompt:

> Write a game similar to Battle City in a single HTML file without dependencies, running on desktop and mobile.

The latest free versions of the following AI chats were used:

1. ChatGPT https://chatgpt.com/
2. Gemini https://gemini.google.com/
3. Cursor IDE https://cursor.com/
4. Copilot from VS Code IDE https://code.visualstudio.com/
5. DeepSeek https://chat.deepseek.com/
6. Claude https://claude.ai/
7. GigaChat https://giga.chat/
8. Alice AI https://alice.yandex.ru/
